@* https://www.playframework.com/documentation/2.6.x/ScalaTemplates *@ @* This template takes three arguments: * * title: a string for the title * head: HTML content to be put in the head * content: HTML content which is the main content of the webpage * *@ @(title: String)(head: Html)(content: Html) @* The 'title' argument gets inserted here. *@ @title @* The stylesheet and favicon get loaded here. *@ @* Any other head tags come last *@ @head @* The page's main content will be inserted here. *@ @content
